Markdown Note

发表于 2017-11-29 | 分类于 Tool

标题

标题前加#和空格，根据#个数不同区分字号，#越多字号越小，貌似最多能有6个。效果略。

粗体和斜体

文字前后加相同个数的*号，1*为斜体，2*为粗体，3*又粗又斜。效果如下：
粗体文字
斜体文字
斜粗文字

引用

要引用的文本内容前加>和空格。效果如下：

川岛和津实凭借清纯稚嫩的天使脸孔、白滑丰满的魔鬼身材，被喻为“20世纪最后の美少女”、“永远的女神”、“纯绝AV界之天使”。

列表

无符号列表项前加-和空格，有符号列表项前加数字.和空格。效果如下：

无符号列表

  • 列表项1
  • 列表项2
  • 列表项3

有符号列表

  1. 列表项1
  2. 列表项2
  3. 列表项3

参考形式的链接

让你可以为链接定一个名称，之后你可以在文件的其他地方使用该链接名称来实现链接。
定义方式为 [链接名称]: 链接地址 “鼠标悬停提示文字”，使用方式为[链接文本][链接名称]，效果如下：
I get 10 times more traffic from Google than from
Yahoo or MSN.

绘制表格

格式如下：
| 第一列 | 第二列 | 第三列 |
| ————- |:————-:| —–:|
| 第一行 | 居中对齐 | 靠右 |
| 第二行 | 居中对齐 | 靠右 |
| 第三行 | 居中对齐 | 靠右 |
第二行设置三列分别为左对齐、居中和右对齐。
